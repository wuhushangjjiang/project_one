## 注释规范

禁止注释中出现制作者的个人信息，如姓名、QQ号、邮箱等

注释尽量简洁明了

当一个HTML板块代码较长时推荐在板块开始时写板块开始注释，板块结束后写板块结束注释

```html
<!-- 活动板块 开始 -->
<div class="part-act">
...
</div>
<!-- 活动板块 结束 -->
```

如果JS需要和后台开发联调的自定义函数时，注明函数的调用方式，函数用途、参数类型、作用等

```js
// 转盘初始化
// 参数1：是奖品的个数，数字类型
// 参数2：用来旋转的圆盘元素，可为dom元素 或 选择器字符串
var panel=new PanelLotery({
    length:8,
    el:'#ltpanel'
});
```

#### 图片命名

1. 图片名称必须小写，禁止使用特殊字符、中文

2. 使用英文或拼音缩写

3. 名称间隔是由"-"符号

4. 命名应该体现图片大概用途

   ####  图片标签

   1. PC端img图片必须填写`width`、`height`、`alt`属性
   2. 移动端必须填写`alt`属性
   3. alt不能为无意义字符，需要能表现出图片的含义，如图片为道具图，则应该为道具的名称

   #### CSS样式命名规范

   class、id都需要小写

   命名使用英文，禁止使用特殊字符、拼音

   class命名语义化

   采用BEM规范命名
